# Smart Attendance System

## 📌 Overview
Smart Attendance System is a Java-based project designed to automate student attendance management. It reduces manual work and improves accuracy.

## 🚀 Features
- Student attendance tracking
- Dashboard view
- Report generation
- Database integration (SQL)
- Web interface (HTML, CSS)

## 🛠️ Tech Stack
- Java
- HTML, CSS
- SQL (Database)
- JDBC

## 📂 Project Structure
- src_Model → Data models
- src_dao → Database operations
- src_Service → Business logic
- web → Frontend (HTML, CSS)

## ▶️ How to Run
1. Clone repository
2. Open in IDE (Eclipse/IntelliJ)
3. Setup database using db_schema.sql
4. Run Main.java
